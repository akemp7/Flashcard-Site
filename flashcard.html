A website to practice JS & jQuery vocabulary!

  • Javascript

    • A programming language to make webpages interactive- not to be confused with Java. Jquery is a library for Javascript that makes it easier to install Javascript for webpages.

  • Operators

    • An operator is a special character that indicates what action is to be performed.

  • Variables

    • A descriptive name for a container. Allows one to label data with a name.

  • Variable naming conventions

    • One should use lowerCamelCase in naming variables in Javascript. Also make them transparent- easy to remember what you mean by a particular variable for you and other programmers.

  • Functions

  • Methods

    • An action run on a piece of data, a sort of "message". But it is different from a function. use the .() notation.

  • Arguments

    • Information the method needs to execute message.

  • Parameters

  • Return

    • This is the methods response.

  • Chaining Methods

    • Involves calling on a method directly on the return of another one.

  • Strings

    • Data type that represents exact text in quotations

  • Booleans

    • True or False Values
